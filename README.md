# Orfeo::API

A [Raku](https://www.raku.org) module to consume Orfeo's REST API.

## Versioning

Orfeo::API uses [Semantic Versioning](https://semver.org).

## Installation

You can install Orfeo::API using [Zef](https://github.com/ugexe/zef).

```
➤ zef install Orfeo::API
```

## Import

```raku
use Orfeo::API;
```
